\doxysubsection{Detailed Description}
Represents a sequential genomic region. 

Allowed to span multiple (sequential) references.

\begin{DoxyWarning}{Warning}
\mbox{\hyperlink{struct_bam_tools_1_1_bam_region}{Bam\+Region}} now represents a zero-\/based, H\+A\+L\+F-\/\+O\+P\+EN interval. In previous versions of \mbox{\hyperlink{namespace_bam_tools}{Bam\+Tools}} (0.\+x \& 1.\+x) all intervals were treated as zero-\/based, C\+L\+O\+S\+ED. 
\end{DoxyWarning}
